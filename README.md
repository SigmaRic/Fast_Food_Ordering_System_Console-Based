# Fast Food Ordering System
**Project Title: Fast Food Ordering System**

**Language: C++**

**Compiler: Dev C++**

## How to Run the Program
1.	Place Project_GR1.cpp and all .txt files in the same folder.
2.	Open Project_GR1.cpp using Dev C++.
3.	Compile and run the program.
4.	Make sure menu.txt is available in the same folder before running.
5.	If some output files do not exist, the system will automatically create them during the first run.

## Text Files Used
**The system uses the following text files:**
1.	menu.txt - stores menu item records
2.	users.txt - stores admin and customer account records
3.	orders.txt - stores order queue records
4.	order_items.txt - stores order item details
5.	order_history.txt - stores completed order history
6.	receipt.txt - stores receipt output
7.	sales_report.txt - stores sales report output
8.	order_summary.txt - stores order summary output
